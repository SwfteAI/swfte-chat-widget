Integration Code

Here's how easy it is to add the chat widget to your website:

// Include the SDK (via CDN or npm) <script src="https://cdn.swfte.com/chat-widget.umd.js"></script> <script> // Initialize the chat widget const client = SwfteChat.createSwfteChatClient({ baseUrl: 'https://api.swfte.com', widgetId: 'your-widget-id', }); // Create and mount the widget const widget = client.createWidget({ position: 'bottom-right', theme: { colors: { primary: '#667eea' } } }); widget.mount(document.body); </script>

Programmatic Control

The SDK provides full programmatic control over the widget. You can open/close the chat, send messages, identify users, and more.

// Identify a logged-in user client.identify({ email: 'user@example.com', name: 'John Doe', metadata: { plan: 'premium' } }); // Listen for events client.on('message:received', (message) => { console.log('New message:', message); }); // Send a message programmatically client.sendMessage('Hello, I need help!');
